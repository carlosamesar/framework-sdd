# Módulos OpenSpec (opcional)

Este directorio es el lugar previsto para el layout ERP numerado:

`openspec/modules/XX-module-<nombre>/changes/YY-<slug>/`

En clones mínimos del Framework-SDD **puede estar vacío**: las especificaciones activas viven en [`../changes/`](../changes/). Cuando se integre el frontend modular o el índice `project.md` completo, crear aquí los módulos y enlazarlos desde `registry.md`.
